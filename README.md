# portfolio

A site that showcases my projects, experience, and all about me!

## Local development

```bash
cd site
npm install
npm run dev
```

## Project structure

- `site/app/` - Next.js app router entry files
- `site/components/layout/` - shared layout primitives
- `site/components/sections/` - page-level sections
- `site/data/` - editable site content/config
- `site/lib/` - reusable constants/utilities
